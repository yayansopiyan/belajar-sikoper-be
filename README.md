# BELAJAR SIKOPER [API]

### Requirement
- PHP min 7.4
- Composer (php package manager)  
- php extension: default, curl, intl, mbstring, php-fpm

### Stack
- Laminas API Tools

### Development

Local
``` bash
# dev server at http://localhost:8081
$ composer development-enable
$ php -S localhost:8081 -t public public/index.php
```
